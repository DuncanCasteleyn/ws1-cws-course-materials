INTRO CWS1

Unobtrusive scripting

Live without Javascript

  • For web pages, you can actually live without Javascript or CSS: HTML/PHP is all you need to display information and get feedback.
  • CSS improves on visual aspects, Javascript on interaction:
    • complex user interaction: drag / drop, keyboard navigation...
    • animations
    • clientside form checking
    • less page reloads thanks to AJAX
    • ...

Javascript as plugin

  • The idea is: just as you plug in a CSS file for design, you plug in a JS file for interactions.
  • Try to see Javascript as an addition to a fully functional page:
    • along with server side formchecking, provide cliendside formchecking
    • on top of classic prev / next HTML links, build a Javascript slideshow
    • replace an existing scrollbar with your fancy Javascript version
    • improve a text field with a color or date picker
    • ...
  • this concept is called unobtrusive scripting and more generally progressive enhancement

Web applications

  • Some web pages have become so rich that we rather speak of web applications:
    • Outlook webversion (IE only, the bastards)
    • Google Docs
    • Javascript games
    • 2D Canvas, OpenGL
    • ...
  • Even in those cases we'll start from a basic version, and check for support before adding features like sound, drag / drop, geolocation, webcam etc...

About this course

OPO Webscripting 1

  • Webscripting 1 = main course (OPO)
  • Contains two sub-courses (OLA's)
    • OLA Clientside Webscripting [JLW321]
    • OLA Serverside Webscripting [JLW322]
  • Each sub-course has a weight of 50%

CWS Course topics (1)

  • What you eventually should know:
    • Fundamentals
      • basic syntax (variables, control structures, functions, scope, closures, ES6...)
      • OO concepts (namespacing, anonymous functions and objects, classes, prototypes...)
      • basic patterns (anonymous objects & functions, self-invoking function, module pattern)
      • advanced patterns (pubsub mediator, facade pattern...)
    • Extensions
      • frameworks (Polymer, AngularJS, NodeJS...)
      • libraries (jQuery & jQuery UI, tinyMCE, lightbox, modernizr, Raphaël...)
      • Ajax, JSON(P), web API's: Google maps, Apps for Ghent...
    • Interaction
      • DOM, querySelector & other DOM navigation functions
      • event handling
      • animations, requestAnimationFrame
      • shadow DOM, web components
    • HTML5
      • storage & caching (localStorage, sessionStorage, WebSQL, app. cache...)
      • API's (audio, file, geolocation, history, device orientation, notifications, contacts, media capture, battery, fullscreen...)
      • multimedia (WebGL, Canvas, SVG)
    • Devtools
      • dependency management (NPM, bower)
      • task automation (Gulp, Grunt)
      • preprocessing (concatenation & minification)
      • code quality (jsHint, editorconfig)
      • Phonegap
